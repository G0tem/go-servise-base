# go-servise-base

## Сервисы

- **PostgreSQL** (`local-postgres-db`) - основная база данных для хранения структурированных данных
- **MongoDB** (`dolocal-mongo-dbcdb`) - документоориентированная база данных для хранения неструктурированных данных
- **RabbitMQ** (`local-message-broker`) - брокер сообщений для асинхронной обработки данных
- **Redis** (`local-redis-server`) - кэш-сервер для временного хранения данных
- **traefik** (`traefik` + `whoami`) - Proxy-сервер для общей документации на 8080
